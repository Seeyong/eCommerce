[네이버 스토어팜]
============
## 1. 발주서 변환 프로그램
> 스토어팜 엑셀 파일을 발주형식에 맞게 변환해주는 프로그램

<br></br>

### ``` 아래 절차에 맞게 사용```
```
Step.1 - 스토어팜에서 발주하려는 업체 상품 체크
Step.2 - '선택주문 엑셀다운로드' 버튼 누름
Step.3 - '다운로드' 디렉토리의 해당 엑셀 파일 이름 Cmd + C(복사)
Step.4 - 프로그램 실행 후, "Insert the file(excel) name :" 가 뜨면
         다운받은 엑셀 파일 이름 넣고 Enter
Step.5 - '데스크탑'에서 발주서로 변환된 엑셀(.xlsx) 파일 확인
```
<br></br>

### ``` 변환 가능한 주요 변수(=변수명)```
```
1. 다운받은 파일 경로 = download_path
2. 자신의 업체명 = my_company
3. 발주하려는 파트너사 업체 리스트(업체명) = partner_list
4. 변환한 파일 생성 경로 = creating_path
```
